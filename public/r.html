ARC-Based Order Flow

1. PR Creation

- Department or employee raises a Temporary PR, reviewed by the Purchase Officer.

2. Budget Check and Approvals

- System checks budget availability, restricting actions if there’s a shortage. - Multi-level PO approvals based on cost.

3. ARC-Based PO

- Syncs with ARC status, enabling streamlined PO creation with minimal input.

4. PO Publishing to Vendor

- POs are published with automated alerts, requiring vendor acknowledgment tracked in the Vendor Portal.

Benefits of ARC-Based Order Flow



1. Streamlined PR Creation

- Simplifies PR creation with departmental input and officer review.

2. Effective Budget Management

- Ensures budget compliance with multi-level approvals.

3. Efficient ARC Utilisation

- Minimizes input by leveraging existing ARC agreements.

4. Enhanced Vendor Communication

- Automated alerts keep stakeholders informed; vendor acknowledgment is tracked in the system.
